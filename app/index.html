Chess API

The dxc4 Chess API allows you to traverse a game's history, apply new moves, and even invoke a chess engine to play. Among other things you can request a FEN string, a board object, the current player's available moves, the game's status and much more. See dxc4.com for an example application that utilizes many of these features.

Check out the demo

The demo chess client utilizes most of the API to create a fully featured chess application without doing any of the heavy lifting. You can import and share games and even invoke the stock fish engine to make moves or to play against.

The source code is open.
