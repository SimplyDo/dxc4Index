How it works

Each request to the API should include a set of moves in algebraic notation, starting from the initial board position. You can set the game's state to a specific move, apply a new move or have the stock fish chess engine make a move.

Once the moves are processed the API will return any requested information. Among other things you can use it to render a board, select and validate next moves, etc...

Send POST requests to: http://api.dxc4.com/pgnparser

For now all the functionality lives in the PGN Parser. See table for optional parameters or better yet explore the live docs to interact with the API directly.

Please note: The pgn parser requires POST requests to accommodate large sets of moves.

Parameter Description
pgn A PGN compatible moves string starting from the initial board position.
Example: 1. d4 d5 2. c4
moveCount Optionally sets the game's state to an earlier move and discards any subsequent moves. Negative numbers will be offset from the last half move.
Example: -3
nextMove A single legal move in algebraic notation to be applied to the game.
Example: dxc4
stockfish Experimental: If set to true the best next move will be determined by the stockfish chess engine and applied to the game.
includes A set of items to be included in the response. Pass in one or more as a comma separated string. See the full list below or consult the live docs for example responses.

API Response

You can request any or all of these via the includes parameter with your requests to the PGN Parser. To explore full responses please consult the live docs.

Name Description
status Returns true/false for: Checkmate, Check, Stalemate, Repetition
side Which player's turn is it?
pgn A PGN compatible moves string starting from the initial board position.
fen Returns a FEN string for the current position. Please note: This currently only includes the actual position and who's turn it is.
valuation A count of all the pieces on the board by type for each side and their standard valuation.
previousMoves An array of all moves in algebraic notation.
previousFullMoves All moves with additional information about the squares and pieces involved.
availableMoves An array of all possible legal moves for the current player in algebraic notation.
availableFullMoves Any possible legal moves for the current player with additional information about the squares and pieces involved.
board Returns an array of all the board's squares and pieces.
